A Simple Soundness Proof for Dependent Object Types

Artifact Guide

Our paper presents a simplified type-safety proof for the version of DOT presented at Wadlerfest 2016. This artifact contains a formalization of the simple proof in Coq. The definitions of the abstract syntax and several lemmas are based on the original Coq proof that comes with Wadlerfest DOT paper.

Description of the Artifact

This artifact presents a Coq formalization of the DOT calculus and DOT's type safety proof using the method presented in the paper. The artifact also provides proofs for Properties 1 (Inert Context Guarantee) and 2 (Tight Typing Guarantee), which are not part of the safety proof, but are discussed in the paper.

The type-safety proof can be found in TODO. The proof of the properties can be found in TODO.

Below, we present a table that shows how the definitions and lemmas in the paper correspond to the artifact. However, before comparing the correspondence, we recommend reading through TODO (Definitions.v). That file not only presents the definitions of the abstract syntax and type system of DOT, but also presents some of the tactics, functions, and inductive definitions that are used in the proof.
