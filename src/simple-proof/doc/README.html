Description of the Artifact

This artifact presents a Coq formalization of the definitions, lemmas, theorems,
properties in the paper, and their proofs. We prove type safety for DOT using
the method presented in the paper, and additionally prove Properties 1 (Inert Context Guarantee)
and 2 (Tight Typing Guarantee), which are not part of the proof,
but are discussed in the paper.

The type-safety proof can be found in TODO.
The proof of the properties can be found in TODO.

Below, we present a table that shows how the definitions and lemmas in the paper correspond
to the artifact. However, before comparing the correspondence, we recommend reading
through TODO (Definitions.v). That file not only presents the definitions of the abstract
syntax and type system of DOT, but also explains other functions and inductive definitions
that are used in the proof.
